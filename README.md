# SI5351-TCXO-Radio

A double-conversion superheterodyne AM radio receiver built around the SI5351 clock generator and a 26 MHz TCXO reference. The design uses three LT5560 active mixers, two BGA2869 LNA stages, and a PAM8406 class-D audio amplifier — all on a single PCB.

## Overview

The receiver follows a classic double-conversion topology:

```
Antenna → RF filter → LNA → Mixer 1 → 10.7 MHz filter → Mixer 2 → 455 kHz filter → LNA → Mixer 3 → Op-amp → Speaker amp
```

The three local oscillator signals are generated by an MS5351M (SI5351-compatible) PLL, locked to a 26 MHz TCXO for frequency accuracy and stability.

## Features

- **Frequency range:** Determined by RF preselector filter (tunable via CT301 trimmer)
- **First IF:** 10.7 MHz (ceramic bandpass filter)
- **Second IF:** 455 kHz (ceramic bandpass filter)
- **LO source:** MS5351M PLL + 26 MHz TCXO, I²C programmable
- **Three mixers:** LT5560 double-balanced active mixers (CLK0, CLK1, CLK2)
- **Two LNA stages:** NXP BGA2869 MMIC (~32 dB gain, 3.1 dB NF)
- **Audio output:** PAM8406 class-D stereo amplifier (2×3 W into 4 Ω)
- **Power supply:** 5 V regulated (signal chain) + dedicated 5 V for audio amplifier
- **Input connector:** SMA (3 GHz rated)
- **ESD protection:** PESD0402-140 on antenna input

## Quick Start

1. Order PCB and components (see [BOM.md](BOM.md))
2. Solder SMD components — start with ICs, then passives
3. Connect 7–12 V DC to the power input connector (U103)
4. Connect an antenna to the SMA connector (RF301)
5. Program MS5351M via I²C with CLK0, CLK1, CLK2 frequencies for your target band
6. Connect a 4–8 Ω speaker to CN1201

## Power Requirements

| Rail | Regulator | Typical current | Purpose |
|------|-----------|----------------|---------|
| +3.3 V | AMS1117-3.3 (U101) | ~34 mA | MS5351M, TCXO |
| +5 V | AMS1117-5.0 (U104) | ~97 mA | LNA, mixers, op-amp |
| +5V_P_AMP | AMS1084CM-5.0 (U102) | ~100–900 mA | PAM8406 audio amp |

Input voltage: **6–15 V DC** (higher input = more heat in regulators; 7–9 V recommended).

## PCB Layout Notes

- AMS1084CM-5.0 (U102) tab pad is **+5V_P_AMP** — not GND. Use an isolated copper polygon.
- AMS1117-3.3 (U101) and AMS1117-5.0 (U104) tab pads are **Vout** — not GND.
- LT5560 exposed pad (EP, pin 9) is **PGND** — connect to GND plane with multiple vias.
- Place decoupling capacitors as close as possible to each IC.
- Use a ground plane on the bottom layer; connect to top layer GND pours via stitching vias.
