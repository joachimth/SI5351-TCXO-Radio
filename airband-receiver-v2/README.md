# AM Airband Receiver V2 — 118–137 MHz

A single-conversion superheterodyne AM receiver designed for the aviation airband (118–137 MHz). This is a simplified, lower-complexity alternative to the triple-conversion SI5351-TCXO-Radio V1 design, retaining the same power supply architecture, audio chain, and LO reference.

**Design philosophy:** Simplicity over perfection. Single IF stage, no baluns, resistive matching, proven building blocks, every stage independently measurable.

---

## Overview

```
Antenna (50Ω SMA)
→ RF Bandpass Filter (118–137 MHz, LC, trimmer-tuned)
→ [Optional attenuator pad]
→ [Optional BGA2869 LNA, bypassable]
→ AD8342 Mixer 1  (RF → 10.7 MHz IF)
→ 10.7 MHz Ceramic BPF
→ AD8342 Mixer 2  (product detector, 10.7 MHz → audio)
→ LT6202 op-amp   (audio LPF + gain)
→ PAM8406 class-D amp
→ Speaker
```

Local oscillator frequencies generated by the same **MS5351M PLL + 26 MHz TCXO** used in V1:

| CLK output | Frequency | Function |
|---|---|---|
| CLK0 | 128.7–147.7 MHz (RF + 10.7 MHz) | LO1 — RF to IF conversion |
| CLK1 | 10.7 MHz (fixed) | LO2 — product detector / AM demodulation |
| CLK2 | unused | — |

---

## Features

- **Target band:** 118–137 MHz AM (aviation VHF airband)
- **IF:** 10.7 MHz (single conversion, ceramic BPF)
- **Mixer:** AD8342 active mixer ×2 — low LO drive, single-ended capable
- **LNA:** NXP BGA2869 (optional, bypassable via jumper, 32 dB gain, 3.1 dB NF)
- **AM demodulation:** Product detector (synchronous) — no diode detector
- **LO source:** MS5351M PLL + 26 MHz TCXO, I²C programmable
- **Audio output:** PAM8406 class-D stereo amplifier (2×3 W into 4 Ω)
- **Power supply:** Same three-rail architecture as V1 (3.3 V, 5 V, 5V_P_AMP)
- **Input connector:** SMA

---

## Key Differences from V1

| Feature | V1 (SI5351-TCXO-Radio) | V2 (Airband) |
|---|---|---|
| Target frequency | Tunable (preselector trimmer) | 118–137 MHz fixed airband |
| Conversion stages | Triple (10.7 MHz + 455 kHz) | Single (10.7 MHz) |
| Mixer IC | LT5560 ×3 | AD8342 ×2 |
| Baluns | WBC1-1TLC ×5 | None |
| IF stages | 2 (10.7 MHz + 455 kHz) | 1 (10.7 MHz) |
| IF amplifier | BGA2869 ×1 at 455 kHz | None (add if needed) |
| AM detection | Synchronous (3rd mixer at 455 kHz) | Product detector (2nd mixer) |
| LO outputs used | CLK0, CLK1, CLK2 | CLK0, CLK1 |
| Signal chain ICs | 7 (3 mixers + 2 LNA + PLL + TCXO) | 4–5 (2 mixers + optional LNA + PLL + TCXO) |

## Recommended Bring-Up Sequence

1. Build and verify power supply rails (+3.3 V, +5 V, +5V_P_AMP)
2. Verify SI5351 (MS5351M) I²C communication and CLK output frequencies
3. Build audio chain (LT6202 + PAM8406) — inject test tone at LT6202 input
4. Build product detector (AD8342 Mixer 2 alone) — inject 10.7 MHz AM signal, verify audio output
5. Build 10.7 MHz IF filter — verify passband insertion loss
6. Build RF mixer (AD8342 Mixer 1) — inject 127.5 MHz test carrier, verify 10.7 MHz IF output
7. Build RF bandpass filter — verify 118–137 MHz passband
8. Full chain test — receive a local VOR or AM airband transmission
9. (Optional) add BGA2869 LNA if sensitivity is insufficient

## Known Trade-offs

| Area | Trade-off |
|---|---|
| Ceramic IF filter | Requires resistive impedance matching (~330 Ω) |
| No AGC | Simpler but lower dynamic range than V1 |
| Product detector LO phase | Free-running LO2; phase offset attenuates audio slightly but voice remains intelligible |
| Single conversion | Image rejection relies entirely on RF BPF (image at LO1 − IF) |
| No IF gain | May need IF gain block (e.g. GALI-84+) if sensitivity is insufficient without LNA |
| Single-ended AD8342 | ~3 dB degradation vs fully differential; acceptable for prototype |
